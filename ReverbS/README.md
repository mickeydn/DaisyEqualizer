# ReverbS

## Author

Kim Bjerge

## Description

This project implements a stereo Schroeder Reverb using four comp filters and two all-pass filters.

The SDRAM is used for one-seconds delay buffers.

Potentiometer 1 controls the wet/dry mix.

Potentiometer 2 controls the room size.

Switch 1 turns on the reverb indicated when the LED 1 is on.

LED 2 changes color depending on the room size.
